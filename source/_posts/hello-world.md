这篇文章是博客初始化后的第一篇占位文章。

你可以先从下面几件事开始：

1. 修改站点配置里的标题、作者、域名和社交链接。
2. 替换 `source/_data/avatar/avatar.webp` 为你自己的头像。
3. 删除或改写这篇文章，开始发布你自己的内容。

常用命令：

```bash
npm install
npm run server
npm run build
```

如果你准备部署到 GitHub Pages，这个模板已经补上了工作流。把仓库推到 GitHub 后，打开 Pages 并选择 `GitHub Actions` 作为部署来源即可。
